# Multi-Module-Template

Template for multi module Android Studio project, using Kotlin Gradle DSL

If you want to make a new project based on this template, do the following steps:

1. Change `rootProject.name` in [settings.gradle.kts](settings.gradle.kts)
2. Refactor package name and update the `namespace` in module's `build.gradle.kts`
3. Change `applicationId` in [app/build.gradle.kts](app/build.gradle.kts)
4. Change `app_name` in [app module's res/values/strings.xml](app/src/main/res/values/strings.xml)
5. Update the app icon
